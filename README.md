# PartyGameApp
This is an app that replaces all card games by making a website that has a storage based system where players/users can upload packs of cards/dares/truths/whatever onto the browse section or into the cloud so other users can download and play with those dares. An easier way to switch between games and modes of the party, hangout, pregame, etc. This game is going to start off as a website and if it grows then maybe into an app. The website has a home page where the setup confirms everyone whos playing the game or you could even do random mode which is still in progress. After the person has been selected another wheel will appear with the dares, truths, or shot will appear and under will say the prompt of whatever was landed on. Then you could either spin the wheel again to get another dare for that player or move onto the next and repeat the cycle. This website/game is intended use for hangouts, Pregames, parties, etc...
